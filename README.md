## 🔍 Troubleshooting Common Issues

*   App does not start: Ensure your computer meets the RAM requirements. Close other heavy programs like web browsers before launching the emulator.
*   No sound: Check the sound settings in the Windows taskbar. Make sure the output device matches your speakers.
*   Black screen on launch: This often relates to graphics drivers. Update your graphics card driver from the manufacturer website.
*   VT-x Error: This means Virtualization is off in your BIOS. Restart your computer, enter the BIOS menu during boot, and find the "Virtualization" option to enable it. Save the changes and restart Windows.
